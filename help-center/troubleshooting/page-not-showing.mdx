---
title: Page Not Showing Up
description: Quick fix if documentation page is missing.
---

## Symptoms

- Your documentation page doesn't appear in the menu
- Page returns a 404 error

## Possible Causes

- The page was not published
- It was assigned to the wrong project

## Quick Fix

<Steps>
  <Step title="Check publish status">
    Open the page, ensure "Published" is enabled.
  </Step>
  <Step title="Verify project assignment">
    Confirm the page is under the correct project folder.
  </Step>
  <Step title="Refresh menu">
    Reload the dashboard to update the navigation.
  </Step>
</Steps>
