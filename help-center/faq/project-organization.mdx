## Overview

Aleksandr Kislitsyn's documentation lets you group docs by project.

## Steps

1. Go to the dashboard home.
2. Click "New Project".
3. Name your project and set a description.
4. Add docs to the right project folder.

## Tips

- Use folders for categories
- Add tags for quick searching
- Archive old projects
